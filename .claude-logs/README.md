# Claude Command Logs

Ephemeral artifacts generated by slash commands. **Not version controlled.**

## Structure

```
.claude-logs/
├── plans/{plan-name}/          # Planning sessions
│   ├── plan.md                 # The plan
│   └── ios-research-*.md       # iOS research (if needed)
├── implement/{plan-name}/      # Implementation sessions
│   └── REPORT.md               # Completion report
├── bugs/{timestamp}-{slug}/    # Bug fix sessions
│   └── REPORT.md               # Bug diagnosis + fix report
├── reviews/{timestamp}/        # Review sessions
│   └── REPORT.md               # Review report
├── oracle/                     # Oracle consultation exports
│   ├── oracle-context-*.txt
│   └── oracle-instructions-*.md
├── INDEX.json                  # Master index
└── README.md                   # This file
```

## Session Types

| Type | Location | Contents |
|------|----------|----------|
| Plans | `plans/{plan-name}/` | plan.md, ios-research-*.md |
| Implementations | `implement/{plan-name}/` | REPORT.md |
| Bug fixes | `bugs/{timestamp}-{slug}/` | REPORT.md |
| Reviews | `reviews/{timestamp}/` | REPORT.md |
| Oracle exports | `oracle/` | context + instructions |

## Commands

**View index:**
```bash
cat .claude-logs/INDEX.json | jq
```

**Find sessions:**
```bash
# Recent plans
ls -lt .claude-logs/plans/

# Recent bugs
ls -lt .claude-logs/bugs/

# Reviews
ls -lt .claude-logs/reviews/
```

## Best Practices

1. **Never commit:** All files here excluded via `.gitignore`
2. **Update index:** Commands auto-update `INDEX.json`
3. **Exports:** Oracle exports can be shared externally
